# Release Notes - v1.14.0

**Release Date:** February 27, 2026  
**TIBCO Platform Control Plane Version:** 1.14.0

## Overview

This release contains documentation and scripts aligned with **TIBCO Platform Control Plane version 1.14.0** for deploying TIBCO Platform on Azure Kubernetes Service (AKS) clusters.

## What's Included

### Documentation
- Complete setup guides for AKS cluster deployment with Control Plane and Data Plane
- Instructions for hybrid deployments (SaaS Control Plane with AKS Data Plane)
- Observability setup guides with Prometheus and Elastic Stack (ECK)
- Azure DNS configuration and certificate management guides
- Storage configuration for Azure Disk and Azure Files

### Scripts
- AKS cluster creation automation
- PostgreSQL and MailDev setup scripts
- Certificate generation and management
- Azure DNS configuration automation
- Environment variable configurations

## Key Features

- ✅ Full AKS cluster setup with TIBCO Platform CP + DP
- ✅ Hybrid cloud deployment support (SaaS CP + AKS DP)
- ✅ Comprehensive observability with Prometheus and ECK
- ✅ Wildcard DNS and certificate management with Azure DNS
- ✅ Azure storage integration (Azure Disk and Azure Files)
- ✅ Multi-cluster deployment support

## Compatibility

- **TIBCO Platform Control Plane:** v1.14.0
- **Azure Kubernetes Service:** 1.24+
- **Kubernetes:** 1.24+
- **Helm:** 3.8+
