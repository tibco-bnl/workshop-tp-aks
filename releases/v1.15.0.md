# Release Notes - v1.15.0

**Release Date:** March 10, 2026  
**Last Updated:** March 10, 2026  
**TIBCO Platform Control Plane Version:** 1.15.0

## Overview

This release contains documentation and scripts aligned with **TIBCO Platform Control Plane version 1.15.0** for deploying TIBCO Platform on Azure Kubernetes Service (AKS) clusters.

## What's New in 1.15.0

### Key Changes from v1.14.0

#### Control Plane Deployment
- **Unified Chart Deployment**: Simplified deployment using `tibco-cp-base` chart
- **Enhanced Secret Management**: New required secrets for session keys and encryption
- **Improved Network Policies**: Better namespace labeling and network isolation
- **Helm 3.13+ Required**: New label-based deployment tracking (`--labels layer=number`)
- **Certificate Management**: Separate certificates for `my` and `tunnel` domains recommended

#### Data Plane Deployment
- **Updated Infrastructure Chart**: New `dp-core-infrastructure` chart with enhanced capabilities
- **Kong Ingress Support**: Optional Kong ingress controller for user applications
- **Enhanced Observability**: Improved Elastic Stack and Prometheus integration
- **Better Storage Configuration**: Enhanced Azure Files and Disk storage class options

#### New Features
- **Developer Hub**: Updated to version 1.15.14
- **BWCE and Flogo**: Updated provisioners with new capabilities
- **Observability Service**: New version 1.15.19 with improved monitoring
- **Event Processing**: New addon for event-driven architectures

### Breaking Changes
- Minimum Helm version increased to 3.13.0 (for label support)
- Network policy configuration changes
- Certificate structure changes (separate `my` and `tunnel` domains)
- Updated environment variable naming conventions

## Upgrade Path

### Upgrading from v1.14.0 to v1.15.0

The TIBCO Platform team provides an automated upgrade script for upgrading from 1.14.0 to 1.15.0.

**Upgrade Script Location**: 
```bash
# Clone or pull latest tp-helm-charts repository
git clone https://github.com/TIBCOSoftware/tp-helm-charts.git
# or
cd tp-helm-charts && git pull

# Navigate to upgrade scripts
cd scripts/1.15.0

# Run interactive upgrade
./upgrade.sh
```

**Upgrade Documentation**: 
- [TIBCO Control Plane Upgrade Assistant (1.14.0 → 1.15.0)](https://github.com/TIBCOSoftware/tp-helm-charts/blob/main/scripts/1.15.0/README.md)
- [Official Upgrade Documentation](https://docs.tibco.com/pub/platform-cp/1.15.0/doc/html/Default.htm#Installation/upgrade-control-plane.htm)

### Important Upgrade Notes
- Review the [breaking changes](#breaking-changes) section above
- Backup your current values files before upgrading
- Test upgrade in non-production environment first
- Plan maintenance window for production upgrades

## Compatibility

- **TIBCO Platform Control Plane:** v1.15.0
- **Azure Kubernetes Service:** 1.32+
- **Kubernetes:** 1.32+ (CNCF certified)
- **Helm:** 3.13+
- **PostgreSQL:** 16.x

## System Requirements

### Minimum AKS Cluster Specifications

#### Control Plane Cluster
- **Node Count**: 3+ worker nodes
- **Node Size**: Standard_D8s_v3 or higher (8 vCPUs, 32GB RAM)
- **Total Resources**: 24+ CPU cores, 96+ GB RAM
- **Kubernetes Version**: 1.32+ 
- **Storage**: Azure Disk (Premium_LRS) + Azure Files

#### Data Plane Cluster
- **Node Count**: 2+ worker nodes
- **Node Size**: Standard_D4s_v3 or higher (4 vCPUs, 16GB RAM)
- **Total Resources**: Based on workload
- **Kubernetes Version**: 1.32+
- **Storage**: Azure Files + Azure Disk
